# Support Ticket Classification System

A Machine Learning and NLP-based Support Ticket Classification Dashboard built using Streamlit and Python.

## Features

* Automatic Ticket Classification
* Priority Prediction (High, Medium, Low)
* Support Team Assignment
* Suggested Resolution Generation
* Interactive Dashboard
* Business Insights & Analytics
* Ticket Management System
* Real-Time Predictions

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Natural Language Processing (NLP)
* TF-IDF Vectorization
* Matplotlib

## Dataset Used
Dataset Used

IT Service Ticket Classification Dataset (Kaggle)

Dataset Link:
https://www.kaggle.com/datasets/adisongoh/it-service-ticket-classification-dataset

**Note:** The dataset contains customer support tickets from multiple IT service categories and was preprocessed for machine learning model training.

## Project Explanation

The Support Ticket Classification System automatically analyzes customer support tickets and classifies them into predefined categories using Machine Learning and Natural Language Processing techniques.

The system predicts the ticket category, assigns the appropriate support team, determines ticket priority levels, and provides resolution suggestions to reduce manual effort and improve support efficiency.

## Business Value

This system provides several operational benefits:

### Faster Ticket Routing

Automatically assigns tickets to the appropriate support team, reducing manual workload.

### Improved Response Time

Helps support teams identify and prioritize critical issues quickly.

### Efficient Resource Management

Allows organizations to allocate technical resources based on ticket categories and priority levels.

### Enhanced Customer Support

Reduces ticket resolution delays and improves customer satisfaction.

### Automation of Support Operations

Minimizes repetitive manual classification tasks through AI-driven automation.

### Better Decision-Making

Provides insights into common support issues and service trends.

## Business Scenario

This project was developed for organizations handling large volumes of IT support requests.

The system helps support teams automatically classify incoming tickets, prioritize urgent issues, assign responsibilities to relevant departments, and provide initial resolution guidance.

By automating ticket categorization and prioritization, organizations can reduce response times, improve operational efficiency, and deliver better customer service experiences.

The dashboard transforms support ticket data into actionable insights, enabling organizations to streamline helpdesk operations and optimize support workflows.
